# Eco-Synthetic: ai-patient-simulator (Project Space)

**Customer:** Adam (external).
**Product:** Multilingual, LMS-integrated AI clinical-skills simulation platform for
psychotherapy / LI-CBT / ACT training. Students hold simulated therapy sessions with AI
"patients," get competency-based rubric feedback tied to transcript evidence, and develop
over time. Institutional layer: College -> Programme -> Course -> Simulation -> Attempt,
with role/scope access, dashboards, Canvas + Moodle LTI, grade sync, deterministic technical
support, academic-safety protections, cost governance, and longitudinal analytics.

reference customer and co-shapes the product; reduced-fee or equity terms TBD (A1 + Legal).
**Phase:** Discovery + viability (pre-build). No build committed.
