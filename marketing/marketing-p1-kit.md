## 1. Brand basics (Hila's draft options, for A1)

**Essence.** A real company run by AI agents, owned by one human. Credible, modern, transparent (build-in-public), technical but human. No hype, no unbacked claims.

**Logo concept options** (Hila to render and present):
- A -- Org-node mark: a network/dot-and-line glyph echoing our org chart. (Strongest tie to what we are.)
- B -- ES monogram: interlocking E/S as one continuous loop.
- C -- Eco-circuit: organic form crossed with circuit lines. Risk: "eco" can misread as environmental -- flagged.

**Color palette option:** primary ink-blue `#12203A`, accent synthetic-teal `#2DD4BF`, signal amber `#F4B740`, off-white `#F7F8FA`, slate `#5B6472`, ink `#0C1116`. (Check contrast before publish.)

**Typography option (all free, OFL):** headings Space Grotesk; body Inter; mono accent JetBrains Mono / Space Mono.

**Public voice:** confident, plainspoken, transparent, lightly witty; never hypey; never imply customers/revenue/results we don't have.

## 2. Avatar style (Hila's options, for A1)

Goal: one consistent system across all agents, extensible to every new agent at onboarding.
- Option 1 -- Persona: stylized flat-vector character bust per agent.
- Option 2 -- Badge: a circular badge per agent (monogram + role glyph + role color). Most honest (no synthetic human faces), most consistent, cheapest, infinitely extensible.

Suggested spec if Badge is chosen: 512x512 master; brand palette only; one free open-source icon set (Lucide/Tabler, MIT/ISC); background tinted by group so the org reads visually (Owner office indigo, Leadership teal, Staff slate, R&D blue, Customer Success green, Sales amber).
Constraints: no real-person likenesses; no copyrighted IP; any AI image tool via the gate and free (paid A1). Full set is A1 before public use.
